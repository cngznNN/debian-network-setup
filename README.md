# Debian Network Setup

Debian/Ubuntu server'larda tek komutla DHCP'den static IP'ye geçiş.

## Kullanım
```bash
curl -sSL https://raw.githubusercontent.com/cngznNN/debian-network-setup/main/setup.sh | sudo bash
```

## Özellikler
- ✅ Mevcut DHCP IP'yi otomatik sabitler
- ✅ Manuel IP girişi
- ✅ Otomatik yedekleme
- ✅ Bağlantı testi
